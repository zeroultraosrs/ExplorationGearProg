Changelog

Version 1.3.2

  • Tormented bracelet, Ring of suffering, and synapse weapons all moved into the major 80-90slayer block. They are postponed to account for ancient shard scarcity.
  • Bandos hilt is identified as a pointless inclusion and as a result is dropped.
  • Soulreaper axe is removed. Too niche, and left to player discretion.

Version 1.3.1

  • Ring of suffering moved to immediately pre-zulrah.
  • Bandos chestplate and tassets removed. Tassets not impactful enough anymore
  • Spelbook swap moved to pre-ZMI.
  • Deadeye and mystic

Version 1.3.0

  • Oathplate armor added.
  • Rite of vile transference.
  • Neitiznot faceguard removed.

Version 1.2.2

  • Mobile QOL, courtesy of RedSparr0w

Version 1.2.1

  • Changed style of url to make it less harsh on the eyes, per request.

Version 1.2.0

  • Black demon task extension has been correctly substituted with greater demon task extension.
  • Removed Seers ring (i) as it no longer is required for maxhits.
  • Mixed hide and chinning moved one step earlier to match the sequencing in BruhSailer.
  • Right click option for going to wiki has been added.
  • The codebase for the website has been reworked for simpler maintenance and in preparation for a future project.
  • Removed light mode for now. It may make a return in the next update if a sizeable portion of users prefer it.
  • Bare bones and original version now synchronize with respect to item selection.
